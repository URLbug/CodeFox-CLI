CodeFox-CLI
===========

Краткое описание
-----------------
CodeFox-CLI - это интеллектуальная система автоматического код-ревью, которая берет на себя рутину проверки безопасности и качества кода, позволяя Senior-разработчикам фокусироваться на архитектуре и сложных задачах. 
В отличие от обычных линтеров, CodeFox понимает контекст всего проекта и бизнес-логику, предлагая не просто замечания, а готовые исправления (Auto-Fix).

Установка
---------
- Установить зависимости локально:
	pip install -r requirements.txt
- Установить в режиме разработки (локальная команда `codefox`):
	python -m pip install -e .
- Установка из репозитория
    python -m pip install https://github.com/URLbug/CodeFox-CLI.git

Быстрый старт
-------------
- Инициализация (сохранение API-ключа):
	python -m codefox --command init
- Выполнить скан (использует текущий git-diff):
	python -m codefox --command scan
- Посмотреть версию:
	python -m codefox --command version

Конфигурация
------------
- Файл игнорирования: `./.codefoxignore`  пути, которые не нужно загружать в File Store.

Команды
--------
- `init` — локально сохраняет ключ от Gemini и создает внутри текущей директории `.codefoxignore`.
- `scan` — собирает изменения из git-diff, загружает файлы в File Store и отправляет запросы в Gemini.
- `version` — отображает текущую версию CodeFox CLI.
- `--help` — получить справку по флагам.

Примеры
-------
- Запустить скан в проекте:
	python -m codefox --command scan
- Локально установить и использовать команду `codefox`:
	python -m pip install -e .
	codefox --command version
